
# CCNY ME Class Histroy Search Form

  A static website that creates a search form for CCNY Mechanical Engineering courses.
  The User is able to filter courses by
  - Degree
  - Elective course
  - Class name
  - Professor name

  Currently, the database contains six semesters:
  - Spring 2018
  - Summer 2018
  - Fall 2018
  - Spring 2019
  - Summer 2019
  - Fall 2019
